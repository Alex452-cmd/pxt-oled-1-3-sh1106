# SH1106 OLED 1.3" Display for micro:bit

Driver per display OLED SH1106 128x64 compatibile con micro:bit e MakeCode.

## Funzionalità
- Inizializzazione display
- Scrittura testo
- Disegno pixel
- Linee
- Rettangoli
- Rettangoli pieni
- Aggiornamento display
- Pulizia schermo

## Hardware
Display OLED 1.3" SH1106 I2C  
Risoluzione: 128x64  
Indirizzo I2C: 0x3C
